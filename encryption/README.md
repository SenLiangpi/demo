# 专业加密工具 Professional Encryption Tool

一个基于 AES 军事级加密算法的专业文本和文件加密工具。

## ✨ 功能特性

### 🔐 专业加密算法

1. **AES-256-GCM** ⭐ 推荐
   - 256位密钥长度
   - 伽罗瓦计数器模式 (GCM)
   - 提供认证加密
   - 军事级安全标准

2. **AES-192-CBC**
   - 192位密钥长度
   - 密码块链接模式 (CBC)
   - 企业级数据保护

3. **AES-128-CTR**
   - 128位密钥长度
   - 计数器模式 (CTR)
   - 快速加密性能

### 📝 支持功能

- ✅ 文本内容加密/解密
- ✅ 文件加密/解密（.md, .txt）
- ✅ 拖拽上传文件
- ✅ 一键复制结果
- ✅ 加密文件自动下载

## 🚀 使用方法

### 文本加密

1. 切换到"文本加密"标签
2. 选择加密算法（推荐 AES-256-GCM）
3. 输入强密钥（建议16位以上）
4. 输入要加密的文本
5. 点击"加密"按钮
6. 复制加密结果

### 文件加密

1. 切换到"文件加密"标签
2. 选择加密算法
3. 输入密钥
4. 拖拽或选择 .md 或 .txt 文件
5. 点击"加密文件"按钮
6. 自动下载加密后的文件（.encrypted）

### 解密操作

**文本解密：**
- 输入加密文本 → 输入相同密钥 → 点击"解密"

**文件解密：**
- 选择 .encrypted 文件 → 输入相同密钥 → 点击"解密文件" → 下载原文件

## 🔒 安全建议

### 密钥设置

- ✅ 长度至少 16 位字符
- ✅ 包含大小写字母、数字和特殊字符
- ✅ 避免使用常见单词或生日
- ✅ 每个重要文件使用不同密钥

### 密钥管理

- 🔐 妥善保管密钥，密钥丢失将无法解密
- 🔐 不要通过不安全渠道传输密钥
- 🔐 建议使用密码管理器存储密钥
- 🔐 定期更换重要文件的密钥

## 💡 应用场景

- 📄 重要文档加密存储
- 💼 敏感信息传输保护
- 📝 个人笔记隐私保护
- 🔏 源码文件加密备份

- 请勿用于违法用途
- 建议重要数据多地备份
